# やったこと記録アプリ（iOSアプリ）

## 使用言語
Swift5(SwiftUI)

## アプリ仕様
- やったことを記録するアプリ（タイムログ）
  - やった内容と開始／終了時刻を入力して記録
    - 時分単位で入力
  - やった内容を一覧で表示
    - 削除可能（スワイプで削除ボタンを表示）
  - 記録データはSQLite3にて端末内で保持
  - CSV形式で共有可能
